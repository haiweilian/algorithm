# 递归

递归的过程中形成一个系统栈，分为*递*和*归*的过程， 函数调用之前的代码是*递*的过程执行的，函数调用之后的代码是*归*的过程执行的。

# 回溯

如执行时形成一个路径，当不能满足条件的时候再一步一步的回退回去 `做操作 -> 递归 -> 撤销操作`。

## 回溯算法解题通用套路

```js
result = []
function backtrack(路径, 选择列表) {
    if 满足结束条件:
        result.add(路径)
        return

    for 选择 in 选择列表:
        做选择
        backtrack(路径, 选择列表)
        撤销选择
}
```
